# Программа «Денежная копилка»


## Описание

Консольное приложение “Денежная копилка” для управления накоплениями на языке Python, в котором пользователь может
добавлять и удалять цели, отслеживать прогресс по накоплениям, редактировать данные,
а также управлять балансами целей и историей изменений.

## Функциональные возможности

- Добавление и удаление цели.
- Пополнение баланса цели.
- Снятие средств с цели.
- Просмотр прогресса по цели.
- Просмотр всех целей.
- Общий прогресс по всем целям.
- Уведомления о прогрессе при достижении более 50%.
- Напоминания о сроках.
- Расчёт даты завершения цели.


1.1 Добавление цели.
Создание класса Goal, который будет содержать такие атрибуты,
- название цели,
- итоговая сумма, 
- текущий баланс, 
- категория и 
- статус.

Для реализации можно использовать словарь или список для хранения данных о целях.
